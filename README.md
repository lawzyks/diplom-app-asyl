# АПК · Автоматизация дипломов и приложений

Упрощённая веб-версия системы для автоматического заполнения и печати
дипломов и приложений к ним (документ государственного образца РК).
Реализована по разобранному функционалу программы KostanaySoft и вебинара.

Данные учебного заведения и специальности предзаполнены по
**Алматинскому политехническому колледжу** и редактируются прямо в интерфейсе.

## Возможности

- **Связанные справочники**: учебное заведение → специальности →
  квалификации → дисциплины (учебный план) → студенты. Удаление каскадное.
- **Модуль ввода оценок**: экзаменационная ведомость студента по учебному
  плану его специальности, автоматический подсчёт среднего балла.
- **Генерация печатной формы**: диплом и приложение к диплому
  государственного образца, готовые к печати на A4 (`Печать` → PDF/принтер).
- Подсказка о праве на диплом с отличием.

## Технологии

- Фронтенд: **React 19 + Vite**.
- Данные: **имитация базы данных** — реактивный репозиторий поверх
  `localStorage` (`src/data/db.js`). Слой абстрагирован: его можно заменить
  на REST-вызовы к бэкенду Node.js/Express без изменения компонентов.

## Запуск

```bash
npm install      # зависимости уже установлены в node_modules
npm run dev      # режим разработки  → http://localhost:5173
npm run build    # production-сборка → dist/ (готова к развёртыванию)
npm run preview  # предпросмотр собранной версии
```

Сборка `dist/` — статическая, разворачивается на любом веб-сервере или
статическом хостинге.

## Структура

```
src/
  data/        seed.js (данные АПК), db.js (имитация БД)
  lib/         util.js (оценки, средний балл, формат), router.jsx
  components/  ui.jsx, ResourcePage.jsx, Layout.jsx
  pages/       Dashboard, Institution, Specialties, Qualifications,
               Disciplines, Students, Grades, Document
```

## Порядок работы

1. Раздел **«Учебное заведение»** — проверьте реквизиты колледжа.
2. Заполните справочники: специальности → квалификации → дисциплины.
3. Внесите выпускников в раздел **«Студенты»**.
4. **«Ввод оценок»** — выставьте экзаменационную ведомость.
5. **«Печать диплома»** — выберите студента и распечатайте документ.

Кнопка **«Сбросить демо-данные»** возвращает базу к исходному набору.
